获取 API Key

要开始调用 CodexSurf 的 GPT 服务，第一步就是在控制台创建你的专属 API Key。整个流程很短，通常几分钟内就可以完成。

开始前提醒

  • 生成后的 sk-xxxxxx 格式字符串就是你的实际调用密钥，请妥善保存。
  • 建议按工具用途区分不同令牌，后续排查问题和管理额度会更方便。
  • 如果你准备给 CodeX 使用，请确认令牌分组选择的是对应的专用分组。

创建步骤

访问 CodexSurf 控制台，然后按下面的顺序创建：

  1. 进入 「API 密钥」 页面，点击 「创建密钥」。
  2. 名称： 按你的使用场景填写，方便后续区分。
  3. 令牌分组： 根据实际用途选择，属于必填项，不能留空。
  4. 额度建议： 如果你需要长期使用，通常建议设置为无限额度。
  5. 其余选项保持默认即可，确认后完成创建。

使用建议

创建完成后，建议立刻复制并保存到密码管理器或本地安全位置。后续你可以继续前往 CodeX 配置、CC-Switch 使用 或 OpenClaw 配置 页面，将该令牌填入对应工具配置中。
