Node.js 前置安装

CodeX CLI 和 OpenClaw 都依赖 Node.js 环境。开始这两类工具配置前，请先完成本页的安装步骤。

哪些文档需要先装 Node.js

  • CodeX配置： CodeX 通过 npm 安装，必须先准备好 Node.js 环境。
  • OpenClaw 配置： 如果你通过 npm / pnpm 安装 OpenClaw，需要 Node.js 22 或更高版本。

Windows 安装

方法一：使用官方安装包（推荐）

  1. 访问 https://nodejs.org
  2. 下载 LTS 版本的 Windows Installer（`.msi`）
  3. 运行安装程序，按默认设置完成安装
  4. 安装完成后重新打开命令行窗口

方法二：使用包管理器

使用 Winget：

powershell 
winget install OpenJS.NodeJS.LTS

使用 Chocolatey：

powershell 
choco install nodejs-lts

使用 Scoop：

powershell 
scoop install nodejs-lts

macOS 安装

方法一：使用 Homebrew（推荐）

如果尚未安装 Homebrew：

bash 
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安装 Node.js：

bash 
brew install node

方法二：使用官方安装包

  1. 访问 https://nodejs.org
  2. 下载 LTS 版本的 macOS Installer（`.pkg`）
  3. 运行安装程序，按默认设置完成安装

Linux 安装

Ubuntu / Debian：

bash 
sudo apt update
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

CentOS / RHEL / Fedora：

bash 
sudo dnf install nodejs npm
# 或者
sudo yum install nodejs npm

Arch Linux：

bash 
sudo pacman -S nodejs npm

验证安装

bash 
node --version
npm --version

版本建议

CodeX 建议使用 Node.js LTS。OpenClaw 如果通过 npm / pnpm 安装，建议使用 Node.js 22 或更高版本。
